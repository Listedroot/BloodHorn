BloodHorn Bootloader

BloodHorn is a lightweight, modular, and architecture-independent bootloader built from scratch with simplicity, speed, and portability in mind. Written in C and based on EDK2, it provides a clean UEFI-first environment, supporting multiple executable formats and modern development practices.

Created to offer a serious alternative to existing bootloaders, BloodHorn is built for developers who demand control, flexibility, and performance — without compromise.

Features

  • Clean, minimal architecture
  • UEFI native (EDK2 framework)
  • Graphical interface with theme support
  • Multiboot2, ELF, and raw binary loading
  • Architecture-agnostic (x86_64, ARM64, RISC-V, etc.)
  • Modular stage system: Stage 1 / Stage 2 / Stage 3

Project Goal

The goal of BloodHorn is to become the definitive modern bootloader for open source operating systems — small, fast, maintainable, and easy to integrate. It is designed for real-world use on real hardware, with a clear and clean internal architecture.
